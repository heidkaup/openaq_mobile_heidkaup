# OpenAQ Edistynyt mobiiliohjelmointi 2026

OpenAQ (Open AirQuality) on järjestö, joka mm. kerää ilmanlaatuun liittyvää mittausdataa ympäri maailmaa useasta eri maasta, myös Suomesta useista eri paikoista.

Apin teknologiat
- Kotlin
- Jetpack Compose
- Retrofit
- Flask backend

Apin ominaisuudet
- Mittausten listaus
- Detail-näkymä
- Navigointi
- ViewModel arkkitehtuuri

Backend (Flask + MySQL) needs to be running locally for full functionality.

Kaupunki: Helsinki
